# 🌱 Smart Crop Recommendation System

A high-performance Machine Learning web application featuring a premium customized UI. This system evaluates specific soil characteristics and climate conditions to accurately recommend the optimal crop for farming with an accuracy of **99.32%**.

---

## 🚀 Production Deployment
🌍 **Live Application:** [Live](https://crop-recommendation-app-4.onrender.com/)

---

## 📊 Dataset Parameters
The system evaluates 7 critical environmental variables to generate predictions:
* **N (Nitrogen):** Ratio of Nitrogen content in soil.
* **P (Phosphorus):** Ratio of Phosphorus content in soil.
* **K (Potassium):** Ratio of Potassium content in soil.
* **Temperature:** Ambient climate temperature in °C.
* **Humidity:** Relative air moisture content in %.
* **pH:** Soil acidity scale (value ranging from 0 to 14).
* **Rainfall:** Average rainfall measurement in mm.

Supports targeted recommendations across **22 distinct crop varieties** (e.g., Rice, Maize, Coffee, Mango, Kidney Beans, etc.).
